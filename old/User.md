## 操作习惯

### 偏好的交互方式
- [x] 喜欢使用自然语言与 CLI 交互（智能识别模式）
- [x] 期望多步骤任务自动执行，确认后一次性完成
- [x] 重视结果的可视化呈现（表格、图表）
- [x] 每个任务完成后需要 QA 质量检查，确保符合需求

### 命令使用习惯
- 常用中文自然语言输入，如 `sh 分析订单趋势`
- 期望 AI 主动执行而非仅给出建议
- 喜欢简洁的命令，避免过多参数

### 输出偏好
- 表格格式优先
- 中文界面
- 避免 Unicode 特殊字符（Windows 兼容）

## 业务关注点

### 核心指标
1. **订单分析** - 订单量、销售额、客单价
2. **渠道分布** - 各渠道订单占比和趋势
3. **客户分析** - 新客、活跃客户、留存率
4. **地区分布** - 各店铺/区域销售情况

### 分析周期
- 常用周期: 30天、7天
- 关注趋势对比

### 数据导出
- HTML 报告（可打印 PDF）
- 图表可视化
