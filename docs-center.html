Skills Store Docs Center

This page collects the key documents behind the current static prototype so product, design, engineering, and QA can keep iterating with the same structure and design language.

Current Scope

This site is still a static prototype. It includes store browsing, installed skills, developer workspace, admin workspace, and account-related flows. All interactions are demo-only and do not depend on backend APIs.

When expanding the site, follow the existing navigation, card patterns, footer layout, and workspace structure so the whole experience stays aligned with the storefront.

Design language: Colors, shadows, border radii, page skeletons, and component rules.
Page consistency: Top navigation, footer menus, and default page states should match the page filename.
Static boundary: Real login, registration, submission, review, and certificate management logic are not implemented yet.
