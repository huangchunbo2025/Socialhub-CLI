# SocialHub.AI CLI 安装指南

## 前置要求

- Python 3.10 或以上版本

### 检查 Python 版本

```bash
python --version
```

输出 `Python 3.10.x` 或更高即可。如未安装，请从 [python.org](https://www.python.org/downloads/) 下载安装。

---

## 需要哪些文件？

| 场景 | 需要的文件 |
|------|-----------|
| 电脑可以访问互联网 | 只需 `socialhub_cli-0.1.0-py3-none-any.whl` |
| 纯内网、无法访问 PyPI | `socialhub_cli-0.1.0-py3-none-any.whl` + `deps/` 依赖包文件夹 |

---

## 安装（有网络）

将 `.whl` 文件拷贝到本地任意目录，打开终端执行：

```bash
pip install socialhub_cli-0.1.0-py3-none-any.whl
```

pip 会自动从网络下载所需依赖库。如果下载速度慢，可以指定国内镜像：

```bash
pip install socialhub_cli-0.1.0-py3-none-any.whl -i https://pypi.tuna.tsinghua.edu.cn/simple
```

---

## 安装（离线 / 纯内网）

需要将 `.whl` 和 `deps/` 文件夹放在同一目录下：

```
安装包/
├── socialhub_cli-0.1.0-py3-none-any.whl
└── deps/
    ├── typer-...whl
    ├── rich-...whl
    ├── httpx-...whl
    └── ...（其余依赖）
```

执行安装：

```bash
pip install socialhub_cli-0.1.0-py3-none-any.whl --find-links deps/ --no-index
```

> **如何生成 `deps/` 文件夹？** 由管理员在有网络的机器上执行一次：
> ```bash
> pip download socialhub_cli-0.1.0-py3-none-any.whl -d deps/
> ```
> 然后将整个 `deps/` 文件夹和 `.whl` 一起发给同事。

## 快速开始

### 查看帮助

```bash
socialhub --help
```

### 配置别名（可选，推荐）

配置短命令 `sh` 方便日常使用。

**macOS / Linux**（添加到 `~/.bashrc` 或 `~/.zshrc`）：

```bash
alias sh='socialhub'
source ~/.bashrc
```

**Windows PowerShell**（添加到 `$PROFILE`）：

```powershell
Set-Alias -Name sh -Value socialhub
```

配置后 `sh analytics overview` 与 `socialhub analytics overview` 完全等价。

## 常用命令

### 数据分析

```bash
# 业务概览（GMV、订单、AOV、活跃买家）
socialhub analytics overview

# 客户分析
socialhub analytics customers

# 留存分析
socialhub analytics retention
```

### 日报工作流

```bash
# 今日日报（GMV / 订单 / AOV vs 7日均值）
socialhub workflow daily-brief

# 最近 7 天汇总
socialhub workflow daily-brief --period=7d

# 导出为 Markdown 文件
socialhub workflow daily-brief --output=report.md
```

### 客户查询

```bash
socialhub customers search --keyword 张三
socialhub customers get --id C001234
socialhub customers list
```

### 营销活动

```bash
socialhub campaigns list
socialhub campaigns calendar
```

### 积分 & 优惠券

```bash
socialhub points balance --customer-id C001234
socialhub coupons list
```

### AI 自然语言查询

```bash
socialhub ai chat
```

进入对话后，直接用中文描述需求，例如：

```
> 今天的 GMV 是多少？
> 最近 30 天新增了多少客户？
> 帮我列出本月所有进行中的活动
```

## 常见问题

**Q: 安装后提示找不到 `socialhub` 命令**

Python 的 Scripts 目录可能不在系统 PATH 中。执行以下命令找到安装位置：

```bash
pip show socialhub-cli
```

将 `Location` 对应路径下的 `Scripts`（Windows）或 `bin`（macOS/Linux）文件夹加入 PATH。

**Q: pip 下载依赖时很慢或失败**

切换为国内镜像：

```bash
pip install socialhub_cli-0.1.0-py3-none-any.whl -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**Q: 提示连接数据库失败**

CLI 需要访问公司内网 MCP 数据服务，请确认已连接公司内网（或 VPN）。

**Q: 如何升级版本**

收到新版 `.whl` 后执行：

```bash
pip install --upgrade socialhub_cli-0.x.x-py3-none-any.whl
```
